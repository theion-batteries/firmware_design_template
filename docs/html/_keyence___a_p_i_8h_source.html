25/***** rs232 basic device commands ****/
26/*
27* the commands must be sent to the device using CR as delimeter,
28* and the response must be received also with CR.
29* the controller (arduino) must send one command and wait for
30* the response to send next one.
31* General Mode: During measurement The measurement control commands are accepted.
32Commands such as writing/reading setting values are not accepted
33* Communication Mode: • When the command "Q" "0" "CR" is received, the mode changes to
34the communication mode, and the setting values are written/read.
35• The measurement stops during the communication mode.
36* command | description | Mode | response | ERROR
37--------------------------------------------------------------------------
38* Q0 CR | Changing to the communication mode | communication | same as command | Err-51
39--------------------------------------------------------------------------
40* R0 CR | Changing to the general mode | general | same as command | ERROR
41--------------------------------------------------------------------------
42* MS,01 CR | Measured value output (single) 01-12 | General | MS,OUT01,value CR | ERROR
43--------------------------------------------------------------------------
44--------------------------------------------------------------------------
45* MM,010010000000 CR | Measured value output (multiple) 01-12 ex here out2 and out5| General | MM,010010000000,value[,value,value] CR | ERROR
46--------------------------------------------------------------------------
47* MA CR | Measured value output (all) 01-12| General | MA,value[,value,value] CR | ERROR
48--------------------------------------------------------------------------
49* Timing Diagrams
50* t: time between receiving command and responding: process time
51* Mesrument/control PW: 100ms + number of heads x 100ms : if one head: 200ms if 3 heads 100ms +3*100=400ms
52* Mode Change to General: 600 ms + Number of head expansion units x 750 ms = 600+3*750 = 2850 ms
53* Error Code
54• 50: Command error
55• 51: Status error
56• 60: Command length error
57• 61: Parameter count error
58• 62: Parameter range error
59• 88: Timeout error
60• 99: Other error
61*/
